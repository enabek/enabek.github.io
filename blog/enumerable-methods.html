Enumerable Methods

The .each Iterator

When working with arrays and hashes you will quickly run into a problem where you must tap into the container's contents in order to work with them individually. This where the Enumerable method .each will save the day! .each will grab the index, key, and/or value and perform whatever block of code you set it to do. Let's take a look at what iterating with .each actually looks like on arrays and hashes.

Let's say we have an array:

favorite_teas = ["Oolong", "Green Tea", "White Tea"]

If you wanted to print a list of these teas you would iterate in the following way:

favorite_teas.each {|tea| puts tea }

The "tea" item between the brackets is simply a keyword used as a placeholder. You can use any word!

How about hashes? Hashes differ because they now have two placeholders, the key and its value.

tea_stock = ["Oolong" => 5, "Green Tea" => 3, "White Tea" => 3]

If you wanted to print how many teas you have left you would do it as follows:

tea_stock.each {|tea, quantity| puts "#{tea}: #{quantity}"}

Once again, you can use any placeholder for the keywords!
