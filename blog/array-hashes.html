Arrays & Hashes

Ruby Containers

::: Arrays :::

Arrays are ordered collections of objects. They can hold just about any type of variable inside, even other arrays. Arrarys themselves are also considered objects.

my_array = ["object_1", "object_2", "object_3"]

There are many ways to manipulate the data inside the arrays. To access objects within arrary it is important to remember the index positioning of objects. For example, if we wanted to access "object_1" we would call it by using my_array[0] because the beginning index number is always zero.

::: Hashes :::

Hashes are a little more complex because they can hold collections of paired data called key/value pairs. Hashes are useful when you are looking to store pairs of information. They key always points to the value and is used as the index when calling values in hashes.

my_hash = {"key_1" => "value_1", "key_2" => "value_2", "key_3" => "value_3"}
