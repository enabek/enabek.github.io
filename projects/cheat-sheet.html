# code

The break statement is used to terminate a method with an associated block if called within the block (with the method returning nil).

The next statement jumps to next iteration of the most internal loop. Terminates execution of a block if called within a block (with yield or call returning nil).

The redo statement restarts this iteration of the most internal loop, without checking loop condition.
